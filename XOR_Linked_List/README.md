# XOR Linked List

A C implementation of an XOR linked list, a memory-efficient variation of a doubly linked list.

## Overview

An XOR linked list stores the XOR of the previous and next node addresses in a single link field. This allows the list to be traversed in both directions while using less memory than a traditional doubly linked list.

## Features

* Stores integers in sorted order
* Inserts new values into the correct position
* Deletes matching values from the list
* Traverses the list in ascending or descending order
* Demonstrates pointer manipulation using XOR operations

## Project Files

| File                | Description                                     |
| ------------------- | ----------------------------------------------- |

## Program Behavior

The program creates a list of random integers, displays them in ascending order, allows the user to delete values, and then displays the remaining list in descending order.

## Note

This project is intended as a data structures exercise and demonstrates low-level pointer manipulation in C.
