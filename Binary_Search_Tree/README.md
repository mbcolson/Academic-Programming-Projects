# Binary Search Tree

A C++ implementation of a Binary Search Tree (BST) for storing and managing integer key/value data.

## Features

* Insert data by key
* Retrieve data by key
* Delete nodes by key
* Check if the tree is empty
* Calculate tree depth
* Traverse the tree using:

  * In-order
  * Pre-order
  * Post-order

## Files

```text
BSTree.h        # BSTree class and Data structure
BSTree.cpp      # BSTree implementation
BSTreeMain.cpp  # Demo/test driver
```

## Notes

* Keys and stored values are integers.
* Duplicate keys are not inserted.
* The tree is not self-balancing.
* In-order traversal visits keys in sorted order.
