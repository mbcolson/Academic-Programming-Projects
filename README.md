# Programming Projects

A collection of academic and personal programming projects completed while studying computer science.

## Overview

This repository contains projects covering data structures, object-oriented programming, operating systems, computer organization, simulations, and GUI development. The projects are written primarily in C, C++, Java, and Verilog.

## Projects

| Project                               | Topic                                   |
| ------------------------------------- | --------------------------------------- |
| `AVL_Tree`                            | Self-balancing binary search tree       |
| `Binary_Search_Tree`                  | Binary search tree implementation       |
| `Bouncing_Ball`                       | Animation or simulation project         |
| `Calculator_GUI`                      | Graphical calculator application        |
| `Coin_Flipping_Simulation`            | Probability simulation                  |
| `Computer_Organization_Term_Project`  | Computer organization / Verilog project |
| `Heap`                                | Heap data structure                     |
| `Intercepting_Linux_System_Calls`     | Operating systems / Linux system calls  |
| `Linked_List`                         | Linked list implementation              |
| `Matrix`                              | Matrix class or matrix operations       |
| `Operating_Systems_Concepts_Programs` | Operating systems exercises             |
| `Rational_Class`                      | Rational number class                   |
| `String_Class`                        | Custom string class                     |
| `Threaded_Binary_Search_Tree`         | Threaded binary search tree             |
| `Tic_Tac_Toe`                         | Tic-tac-toe game                        |
| `Vector_Class`                        | 3D vector class                         |
| `XOR_Linked_List`                     | XOR linked list implementation          |

## Topics Covered

* Data structures

* Pointer manipulation
* Operator overloading
* GUI programming
* Simulations

## Note

These projects were created through a mix of coursework and independent programming practice. They are intended to demonstrate programming concepts, data structure implementations, and problem-solving techniques.
